# 📰 News App (SwiftUI)

A modern iOS News App built using **SwiftUI + MVVM**, supporting search, offline caching, and unit testing.

---

## ✨ Features

- 🔍 Search articles
- 📱 Universal app (iPhone + iPad)
- 🔄 Pull to refresh
- 💾 Offline mode (SwiftData caching)
- 🖼 Async image loading with placeholder
- ⚡ Smooth list performance
- 🧪 Unit testing (ViewModel + API)

---

## 🏗 Architecture

- MVVM (Model-View-ViewModel)
- Repository Pattern
- Dependency Injection
- Protocol-based networking

---

## 🛠 Tech Stack

- SwiftUI
- Swift Concurrency (async/await)
- SwiftData (Persistence)
- URLSession
- XCTest (Unit Testing)
